# マトリックスのあの画面ジェネレーター

Canvasでマトリックス風の文字雨を生成できる静的Webツールです。好きな文字列やプリセットを使って、画面表示、画像出力、動画素材化ができます。

## 主な機能

- 文字プリセット切り替え
- カスタム文字入力とカスタムプリセット保存
- 文字サイズ、速度、密度、残像、発光、奥行きの調整
- 方向、文字順、ばらつき分布の変更
- PNG出力
- GIF出力
- MP4/WebM動画出力
- JSON設定保存・読込
- フルスクリーン表示

## 使用技術

- HTML
- CSS
- Vanilla JavaScript
- Canvas API
- MediaRecorder API
- localStorage

## 使い方

1. `index.html` をブラウザで開く、または GitHub Pages の公開URLへアクセスします。
2. 設定パネルで文字、速度、密度、発光、奥行きなどを調整します。
3. 画面をクリックまたはタップすると設定パネルを非表示・再表示できます。
4. `PNG`、`GIF`、`MP4/WebM` ボタンから素材を書き出せます。
5. `JSON save` / `JSON load` で設定を保存・復元できます。

## 工夫した点

- 文字入力は改行ごとに別パターンとして扱い、複数の文字列の雨を生成できます。
- 半角英数字を全角化して、日本語文字と横位置が揃いやすいようにしています。
- 描画ループ中のDOM参照を減らすため、設定値をキャッシュしています。
- 高い発光・ぼかし設定でも重くなりすぎないよう、内部の描画半径を圧縮しています。
- 静的ページとして公開しやすいよう、外部script依存を追加せずに実装しています。
