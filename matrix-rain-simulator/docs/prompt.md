# Prompt Log - マトリックスのあの画面ジェネレーター

## 概要

Codex Desktop を使って作成したマトリックス風の文字雨エフェクトジェネレーター。
静的ウェブページとして実装。

## 記録方針

- 変更を加えた場合は、このファイルの「追加・修正の経緯」に追記する。
- セキュリティ上問題になり得る詳細、秘密情報、悪用手順、内部環境に依存する情報は記載しない。

## 追加・修正の経緯

1. **WAR プリセット追加**
   - 落とす文字に「きのこ」「たけのこ」のプリセットを追加。

2. **文字発光の強化**
   - 文字発光の調整幅を拡大。
   - 高い発光値でも描画が破綻しにくいように調整。

3. **梵字プリセットの表示改善**
   - 梵字が表示されやすいよう、対応フォントを利用する構成に調整。

4. **動画エクスポート機能の追加**
   - MP4/WebM 形式で動画を書き出せる機能を追加。
   - スクリーンセーバー素材や背景素材として使いやすい出力にした。

5. **動画の長尺対応**
   - 動画出力の秒数を 5 / 10 / 15 / 30 / 60 秒から選べるようにした。
   - GIF の秒数設定とは分離。

6. **タイトル変更**
   - 表示名を「マトリックスのあの画面ジェネレーター」に変更。

7. **公開向けの安全性調整**
   - 公開ページとして扱いやすいよう、外部依存とブラウザ側の制約を見直した。

8. **作業記録の運用追加**
   - 今後の変更はこのファイルに追記する運用にした。
   - セキュリティ上問題になり得る内容は記録しない方針を明記。

9. **作業記録ファイルの配置変更**
   - 作業記録を `matrix-rain-simulator/docs/prompt.md` に配置するように変更。

10. **先頭文字フラッシュのタイミング調整**
    - 四角い光が先頭文字の現在位置に出るように調整。

11. **文字発光の負荷軽減**
    - 高い発光値でも処理が重くなりすぎないように描画負荷を調整。

12. **毎フレーム処理の軽量化**
    - 設定値の読み取りをキャッシュし、描画ループ中の余計な処理を減らした。
    - 文字列生成数の判定に使う値を明示的に渡すように整理。

13. **設定値の可視化と描画負荷の調整**
    - 各設定項目の現在値を画面上で確認できるようにした。
    - 光彩と文字ぼかしを高くした場合の描画負荷を抑えるように調整。
